# Q1. 자바스크립트에서 this 키워드의 사용과 그 특성에 대해 설명해 주세요.
<details>
  <summary>(1) 기본 개념</summary>

  * **현재 실행 중인 코드에서의 '주인 객체'를 가리키는 키워드**
  * **this는 "어디서 선언됐냐"가 아니라 "어떻게 호출됐냐"로 결덩됨**
</details>

<details>
  <summary>(2) 상황별 this 값</summary>

  * **1. 일반 함수 호출**
  ```JavaScript
  function test() {
      console.log(this);
  }
  test();

  👉 브라우저: `window`
  👉 strict mode: `undefined`
  ```

  * **2. 객체의 메서드로 호출**
  * **3. 생성자 함수 / 클래스**
  * **4. 화살표 함수⭐**
  * **5. call / apply / bind**
</details>

<details>
  <summary>(3) 핵심 정리</summary>

  * this는 **동적 바인딩**
  * **호출 방식에 따라 값이 바뀜**
  * 화살표 함수는 **this를 바꾸지 않고 상위 스코프 것을 그대로 씀**
</details>
